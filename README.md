# PortfolioSanSanivna
Це високопродуктивний інтерактивний сайт-портфоліо, розроблений для демонстрації навичок веб-розробки за допомогою складних анімацій GSAP та динамічної системи відображення проєктів. Сайт вирізняється глибоким UX, багатомовним інтерфейсом та унікальними вертикальними слайдерами для карток проєктів.
