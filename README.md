# Interactive Portfolio Website

Сучасний, високопродуктивний сайт-портфоліо з акцентом на плавну анімацію, інтерактивність та бездоганний досвід користувача (UX).

## Особливості

- **GSAP Animations:** складна послідовна анімація появи елементів, ефекти розмиття (blur) та плавна прокрутка за допомогою `ScrollTrigger`.
- **Dynamic Projects System:** всі проєкти завантажуються динамічно з `projects.json`, що дозволяє легко оновлювати портфоліо без зміни коду.
- **Vertical Image Sliders:** унікальна реалізація вертикального слайдера для кожної картки проєкту, що імітує прокрутку веб-сторінки.
- **i18n Support:** повна підтримка двох мов (UA/EN) із збереженням вибору користувача в `localStorage`.
- **Responsive & Adaptive:** повністю адаптивний дизайн, що ідеально виглядає на мобільних пристроях, планшетах та десктопах.
- **Performance Optimized:** використання властивості `aspect-ratio` для запобігання стрибків верстки та ліниве завантаження (`lazy loading`) зображень.

## Технологічний стек

- **HTML5 & CSS3** (Custom Properties, Flexbox, Grid, Aspect-ratio).
- **JavaScript (ES6+)** (Async/Await, Fetch API, DOM Manipulation).
- **GSAP (GreenSock Animation Platform)** (ScrollTrigger, Timeline).
- **FontAwesome** (для іконок).
- **JSON** (як легка база даних для контенту).
